Ways to add CSS

There are three different ways to add CSS to an HTML page, which are:

  1. INTENAL CSS
    2. Internal CSS is defined in a style block, which will be inside the head section.
       
        <head>
        <style>
                p {
                    color: red;
                }
                <style>
                <head>
  2. EXTRANAL CSS
    3. You can create one file and link to HTML using <link ref="stylesheet" href = "style.css">
  3. INLINE CSS
    4.             <h1 style="color: purple;">I'm harry </h1>

What are CSS Selectors?

CSS selectors allow us to choose specific elements and apply styles to them.

There are different types of CSS selectors, which are as follows:

  1. Universal
  2. Class
  3. id
  4. Element
  5. group
  6. Descendant Selector
  7. Child Selector
  8. Pseudo Selector
